课程内容

  • 第一阶段

    第一阶段

    FIRST STAGE

    java基础

    Java简介、java 类、对象、循环、修饰符、运算符、文件操作

    java面向对象

    继承、封装、多态、接口、抽象、包

    java高级

    java 数据结构、集合框架、范型、序列化、网络编程、多线程

    java 进阶

    java class文件结构、类加载机制、jvm 虚拟机优化

  • 第二阶段

    第二阶段

    SECOND STAGE

    Hadoop生态

    hadoop生态圈介绍：Hadoop、hive、spark、storm、kylin

    hadoop基础

    hadoop环境搭建、hdfs使用、架构设计、yarn的设计、maperreduce结构

    hive

    hive的安装、增、删、改、查、分区、分桶、内置函数udf、hive性能优化

    hbase

    hbase的安装、hbsseshell、hbase region架构、hbase过滤器、hbase高可用

    spark

    scala 语言基础、Spark Core 、Spark sql、Spark Streaming、Spark MLlib

    JStorm

    JStorm安装、JStorm架构、数据流分流合并、事务、Trident、JStorm高可用

  • 第三阶段

    第三阶段

    THIRD STAGE

    用户画像

    通过人口属性、兴趣特征、消费特征、位置特征、设备属性、行为数据等用户基础信息，通过机器学习的方法给用户打标签，通过不同的维度分析用户群体特征涉及技术包括： hadoop、hive、spark、sqoop、kafak、hbase、redis、mysql

    服务数据清洗

    日志数据清洗主要通过，监控后台服务数据ngix产生的用户行为数据，使用高可靠的flume、kafka等数据收集工具，将数据接入hdfs、sparkStreaming进行实时清洗统计处理。涉及技术：sparkstreaming，JStorm,flume,kafka,zookeeper,hbase

    推荐系统

    推荐系统是利用电子商务网站向客户提供商品信息和建议，帮助用户决定应该购买什么产品,以提高网站销售额。目前业界常用的推荐算法包括基于物品、基于用户等协同过滤算法。涉及技术：spark，hadoop，habase，kafka，dubbo

  • 第四阶段

    第四阶段

    FORTH STAGE

    hadoop高级进阶

    Mapreduce数据倾斜优化，yarn应用调度流程、hadoop源码解读，hadoop生产环境部署

    hive高级进阶

    sqljoin优化，数据倾斜，自定义udf，效率优化

    hbase

    hbase 参数调优：海量数据读写，region数据均衡，hbase高可靠使用（副本）、hbase运维

    spark高级进阶

    spark源码走读，sparkyarn模式，sparkstream高可靠使用，spark rdd优化

为什么选择我们？
我们深知大鹏是一家小公司，很渺小，我们要努力的活下去，我们视学员的就业质量和学习成果为我们的生命线。我们所做的一切，都是为了让学员能够掌握当前最先进的大数据知识并切能享受科技带来的乐趣。
我们有什么不同？
我们的办学理念是开放式较远，集全球最先进的大数据知识，融合在一起，知识来源Apache 、github、infoq、csdn，等知名互联网社区结合国内当前最优秀的技术书籍并融入当前企业中最常用的大数据开发技术和流程。
为什么需要报班？
曾几何时，刚入行的的时候也想过报班，但是看到某些培训机构动辄两万的培训费用，望然却步。于是网上各种搜索资料，到某宝购买好多视频，然后一点一点的死扣，没有人知道，没有人交流，命名一个简单的环境问题，按着网上的方式就是实现不出来。那种酸爽相信有过自学技术的人都体会过，自学这条是异常的艰辛，能通过自学找到工作岗位的小伙寥寥无几。除了艰辛之外，更让人困惑的是那么多的知识不知道从何学起，今天从这个blog抄个demo废了九牛二虎之力实现了，明天又按照其他机构的视频做了一个小demo，这样子时间长了，总感觉自己懂一些皮毛，但是面试的时候又不能得到很好表达，总归到底原因有三：1.缺乏系统的培训，基础知识太差，2.缺乏大量的练习，眼高手低，3.没有计划的总结过去以往经验教训，造成废了力气很大却收效甚微。 来到大鹏学习，同学们可以用很低的花费得到老师面对面的指导，有计划有步骤的学习，避免很多不必要的坑，能够和同学们一份分享在学习中遇到的问题，一起快速成长，用最低的花费，华丽转型从事大数据行业。
